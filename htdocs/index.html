Jquery Mobile Notification

jQuery Mobile - Notice Plugin

Notification plug-in for jQuery Mobile.

data-notice：

Success Error

Data- attribute reference：

data-notice success | error
data-notice-title Notice title
data-notice-content Notice content.

Call Method：

$(document).notice('show', {type:'success'}) $(document).notice('show', {type:'error'})

Options：

type success | error
title Notice title
content Notice content.
theme 'notice'
time 1200
speed 300
